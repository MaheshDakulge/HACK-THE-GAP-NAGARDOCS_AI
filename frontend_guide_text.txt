// Tilt toward touch point using Matrix4
GestureDetector(
onTapDown: (details) {
final dx = (details.localPosition.dx / cardWidth - 0.5) * 0.2;
final dy = (details.localPosition.dy / cardHeight - 0.5) * 0.2;
setState(() => _tilt = Offset(dx, dy));
},
onTapUp: (_) => setState(() => _tilt = Offset.zero),
child: AnimatedContainer(
duration: Duration(milliseconds: 200),
transform: Matrix4.identity()
..setEntry(3, 2, 0.001)
..rotateX(-_tilt.dy)
..rotateY(_tilt.dx),
child: folderCard,
),
)
HERO TAG FOR FOLDER → DETAIL TRANSITION
// On folder card:

Login Screen Prompt
Build a Flutter login screen for NagarDocs AI with these exact specs:
FILE: lib/features/auth/presentation/screens/login_screen.dart
ROUTE: /login
DESIGN:
- Dark navy top section (#1A3C5E) with app logo and title
- White card below with Form containing 3 fields:
1. Email (TextFormField, email keyboard)
2. Password (obscured, toggle visibility icon)
3. Department Code (e.g. CSMC-IT)
- Primary button: Login Securely
- Bottom link: Register
ANIMATIONS:
- On screen enter: card slides up from bottom (Offset(0,1) to Offset.zero, 400ms, easeOutCubic)
- On submit: 3D card flip using Matrix4 rotateY, show success state on back
- On error: shake animation using AnimationController + Tween(-8, 8)
STATE: Uses Riverpod StateNotifier from auth_provider.dart
BACKEND: POST /auth/login with {email, password, department_code}
On success: save token to SecureStorage, navigate to /home
On error: show snackbar with error message

Admin Monitor Prompt
Build Flutter Admin Monitor Screen for NagarDocs AI:
FILE: lib/features/admin/presentation/screens/admin_screen.dart
ROUTE: /admin (admin role only — check user.role == 'admin')
3 TABS:
TAB 1 — Presence:
- List of users with avatar circle (initials) + name + designation
- Each user has a PulsingDot widget:
* Green pulsing = online (last_seen < 5 mins)
* Yellow = idle (5-30 mins)
* Gray = offline
- Pulsing: AnimatedContainer width 8px<->10px, 750ms loop,
with BoxShadow spread 0->2 for glow
TAB 2 — Activity Feed:
- AnimatedList with slide-in from right for new items
- Each item: avatar + text + timestamp
- Suspicious items: red background with flash animation
TAB 3 — Security:
- Tamper flagged documents list
- Admin action buttons per user: Suspend / Revoke / Warn / Approve
- Audit log entries in monospace font
POLLING: Timer.periodic every 30 seconds for presence refresh
BACKEND: GET /admin/presence, /admin/activity, /admin/security
POST /admin/ban-user/{id}, /admin/approve-user/{id}
PROVIDER: admin_provider.dart StateNotifier
